== Create a collage of gridded elements through a draggable/resizable interface. ==

EXTENDED_Create a collage of gridded elements through a draggable/resizable interface.

= Compatibility =

This ACF field type is compatible with:
* ACF 5
* ACF 4

== Installation ==

1. Copy the `acf-collage` folder into your `wp-content/plugins` folder
2. Activate the Collage plugin via the plugins admin page
3. Create a new field via ACF and select the Collage type
4. Please refer to the Create a collage of gridded elements through a draggable/resizable interface. for more info regarding the field type settings
